Коротко:
- Вся логика плагинов (.dl) встроена в один основной файл `FPC.py`.
- Папка `libs/` рядом с `FPC.py` используется для хранения библиотек `.dl`. Если её нет или пуста — приложению автоматически копируются встроенные примеры.
- Меню "Библиотеки" позволяет:
  - установить .dl из файла,
  - импортировать .dl из простого текстового файла (.txt) — парсер ищет поля name:{...}, creator:{...}, value:{...}, code:{...},
  - создать .dl через простую форму,
  - просмотреть/включать бинды, применять темы, открывать шаблонные вкладки.
- Bind-лабы ограничены безопасным действием "insert" (вставка текста). Не выполняется произвольный Python-код.

Как использовать:
1. Сохраните `FPC.py` в папку.
2. Запустите: `python editor.py` (рекомендуем Python 3.8+).
3. Меню "Библиотеки" -> "Установить .dl из файла..." / "Импорт .dl из текста (.txt)..." / "Создать новую .dl (форма)...".
4. Файлы .dl — это обычный JSON с полями: `type` (theme|bind|tabs), `name`, `creator`, `value`, `code`.
   - Примеры `.dl` (встроенные) устанавливаются автоматически в папку `libs/`.

Пример структуры `.dl` (JSON):
```json
{
  "type": "theme",
  "name": "Purple Theme",
  "creator": "poteranii",
  "value": "theme",
  "code": {
    "Светлая": {
      "background": "#ffffff",
      "foreground": "#2b0036",
      "cursor": "#2b0036",
      "selectbackground": "#c8a0e0",
      "selectforeground": "#000000",
      "linenumber_bg": "#f4e8f8",
      "tab_bg": "#f7ecfb",
      "tag": {
        "keyword": {"foreground": "#4b0082"},
        "string": {"foreground": "#a31515"},
        "comment": {"foreground": "#008000"}
      }
    },
    "Тёмная": {
      "background": "#1a0b1e",
      "foreground": "#e6d6f2",
      "cursor": "#ffffff"
    }
  }
}
```

Конвертер .txt -> .dl:
- Если у вас есть простой текст в формате:
  ```
  name:{
    Purple_theme
  }
  creator:{
    user123
  }
  value:{
    theme
  }
  code:{
    "Светлая": { "background":"#fff", "foreground":"#000" }
  }
  ```
  используйте "Библиотеки" -> "Импорт .dl из текста (.txt)...", выберите файл. Программа попытается извлечь поля и сохранить `.dl` в `libs/`.

Безопасность:
- Bind-операции: только действие "insert" поддерживается — текст вставляется без выполнения.
- Код внутри `code` для темы/вкладок хранится как данные (JSON), не исполняется.

Как упаковать в .exe (Windows):
- Установите PyInstaller: `pip install pyinstaller`
- Запустите: `pyinstaller --onefile --noconsole editor.py`
- В папке `dist/` появится `editor.exe`. Опция `--noconsole` убирает консольное окно.
