# FPC - Расширяемый текстовый редактор с системой плагинов

FPC (Flexible Plugin-based Code Editor) - это текстовый редактор с модульной архитектурой, поддерживающий систему плагинов в формате `.dl` для настройки внешнего вида и функциональности.

## Основные возможности

- **Система плагинов** - поддержка `.dl` файлов (тематики, горячие клавиши, шаблоны)
- **Безопасная среда** - плагины выполняются в ограниченном режиме
- **Простое управление** - интуитивный интерфейс для работы с библиотеками
- **Кроссплатформенность** - работает на Windows, Linux и macOS

## Архитектура

- **Главный модуль**: `FPC.py` - содержит всю базовую логику редактора
- **Библиотеки**: папка `libs/` для хранения плагинов `.dl`
- **Автоматическая настройка** - при первом запуске создаются примеры плагинов

## Управление плагинами

Через меню "Библиотеки" доступны:
- **Установка** `.dl` из файла
- **Импорт** из текстового формата (`.txt`)
- **Создание** новых плагинов через форму
- **Просмотр** и управление установленными плагинами

## Типы плагинов

### Темы оформления (`type: "theme"`)
Настраивают цветовую схему редактора:
```json
{
  "type": "theme",
  "name": "Purple Theme",
  "creator": "poteranii",
  "value": "theme",
  "code": {
    "Светлая": {
      "background": "#ffffff",
      "foreground": "#2b0036",
      "cursor": "#2b0036"
    }
  }
}
```

### Горячие клавиши (`type: "bind"`)
Добавляют сочетания клавиш для вставки текста (только безопасная операция `insert`)

### Шаблоны вкладок (`type: "tabs"`)
Создают предварительно настроенные вкладки с шаблонным содержимым

## Форматы плагинов

### Основной формат: JSON
`.dl` файлы используют стандартный JSON с обязательными полями:
- `type` - тип плагина
- `name` - название
- `creator` - автор
- `value` - значение
- `code` - содержимое плагина

### Текстовый формат для импорта
Для удобства поддерживается импорт из простого текста:
```
name:{Purple_theme}
creator:{user123}
value:{theme}
code:{"Светлая": { "background":"#fff", "foreground":"#000" }}
```

## Безопасность

- **Ограниченные возможности** - плагины bind могут только вставлять текст
- **Нет выполнения кода** - произвольный Python-код не выполняется
- **Изоляция данных** - код плагинов обрабатывается как данные, не как исполняемый скрипт

## Начало работы

1. **Установка**: Сохраните `FPC.py` в удобную папку
2. **Запуск**: Выполните `python FPC.py` (требуется Python 3.8+)
3. **Настройка**: Используйте меню "Библиотеки" для добавления плагинов

## Сборка в исполняемый файл (Windows)

```bash
pip install pyinstaller
pyinstaller --onefile --noconsole FPC.py
```
Готовый `editor.exe` будет создан в папке `dist/`

## Системные требования

- Python 3.8 или новее
- ~10 МБ свободного места
- Поддерживаемые ОС: Windows 7+, Ubuntu 16.04+, macOS 10.12+

FPC сочетает простоту использования с мощной системой расширений, предоставляя гибкий инструмент для работы с текстом и кодом.
